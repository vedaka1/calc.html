JS. Задание
Задание 1.
Создать страницу, содержащую:
- поле для ввода полного ФИО
- поле ввода даты рождения с помощью удобного поля ввода
-поле для ввода перечня хобби через запятую
-кнопка для выполнения операций
-блоки для вывода информации
-изображений

По нажатию на кнопку:
1) из ФИО изымается ИО, в одном из блоков страницы отображается:
доброе утро/добрый день/добрый вечер, Иван Иванович
/* работа со строками, с объектом дата/время*/

2) по введенной дате определяется количество дней, оставшихся до дня рождения, выводится в один из блоков
/*работа с датой, работа с мат.функциями*/
3) по введенной дате определяется какой был день недели, какой сезон года
в одном из блоков выводится:
ты родился/ты родилась в %день недели%, чудесный осенний/весенний/летний/зимний день
/* работа с элементами даты, работа со строками, с массивами, оператор выбора*/

4) в соответствии со сформированным сезоном в имеющейся на странице картинке отображается весна/осень/зима/лето
/*работа с элементами страницы, работа с массивом*/
5) после закрытия и открытия страницы приветствие сохраняется
/*localStorage*/
6) из поля хобби данные отправляются в массив, сортируются по длине названий, выводится в один из блоков с указанием количества элементов
/*массивы, строки*/
7) добавить на страницу Карусель из фотографий с использованием Bootstrap, содержащую не менее 5 фото и любой компонент на ваш выбор
/*работа с Bootstrap*/
